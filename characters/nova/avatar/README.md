# 모노시안 Avatar Assets

This folder is the avatar mount point for 모노시안.

Current prototype fallback files use a generated monochrome "see-through" style
AI idol sprite inspired by the provided reference image:

- `neutral.png`
- `blank_cute.png`
- `worried_cute.png`
- `sparkle_eyes.png`
- `smirk_cute.png`
- `teary.png`
- `empty_smile.png`
- `see-through.png` (shared transparent source used by the fallback expressions)
- `see-through-source-green.png` (chroma-key source kept for regeneration/debugging)

Later Live2D files can be placed here, for example:

- `nova.model3.json`
- `nova.moc3`
- `textures/`
- `motions/`
- `expressions/`
